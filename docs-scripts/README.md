# Documentation and Scripts

This folder contains all documentation (`.md`) and batch script (`.bat`) files for the DRIMS project.

## Contents

### Documentation Files (.md)
- **README.md** - Main project documentation
- **MOBILE_RESPONSIVE_IMPLEMENTATION.md** - Mobile responsive implementation guide
- **SETUP_COMPLETE.md** - Setup completion notes
- **HOW_TO_START.md** - Getting started guide
- And other project documentation files...

### Script Files (.bat)
- **START_BACKEND.bat** - Start backend server
- **QUICK_START.bat** - Quick start script
- **kill-port-8080.bat** - Kill process on port 8080
- **clear-db-and-start.bat** - Clear database and start
- And other utility scripts...

## Note
All markdown documentation and batch scripts have been organized into this folder for better project structure.
