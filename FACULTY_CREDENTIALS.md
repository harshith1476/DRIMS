## Notes

- All faculty members have the same default password: `faculty123`
- Faculty should change their password after first login (feature can be added)
- Email format: `firstname.lastname@drims.edu` (special characters removed)
- All faculty are in the "Computer Science and Engineering" department
- Each faculty has their publications (conferences, journals, patents) loaded automatically

## Viewing All Faculty

Admin can view all faculty members by:
1. Login as admin
2. Navigate to "Faculty" section
3. All faculty profiles will be visible

## Data Loading

The faculty data is automatically loaded when the backend starts for the first time. If you need to reload:

1. Clear the MongoDB database
2. Restart the backend
3. Data will be loaded automatically
